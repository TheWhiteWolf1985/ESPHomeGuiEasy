7  Supporto Multilingua

7.1  Cambiare lingua

ESPHomeGuiEasy può essere visualizzato in Italiano, Inglese, Tedesco e Spagnolo.

  1. Aprire il menu Impostazioni → Lingua.
  2. Scegliere la lingua desiderata dall’elenco con le relative bandiere.
  3. Confermare; l’interfaccia si aggiorna immediatamente.
L’immagine img/menu_language.png deve mostrare la lista lingue con emoji bandiera.

7.2  Come contribuire a nuove traduzioni

Le traduzioni risiedono in semplici file .json all’interno della cartella Language/. Ogni chiave rappresenta una stringa UI. 

{
  "add_sensor": "Agregar sensor",
  "update_yaml": "Actualizar YAML",
  ...
}
Passi per aggiungere una lingua:
  1. Copiare en.json in xx.json, dove xx è la ISO‑639‑1.
  2. Tradurre ognuna delle voci mantenendo intatte le chiavi.
  3. Aggiungere il codice lingua all’array restituito da Translator.get_available_languages() (se non viene rilevato automaticamente).
  4. Inviare un pull request con il nuovo file.
Importante: evitare di inserire caratteri di ritorno a capo all’interno delle stringhe JSON; mantenere la codifica UTF‑8 senza BOM.
