2  Guida Rapida

2.1  Installazione

Prerequisiti: Python 3.10+ già installato.
  1. Scaricare l’ultima release da GitHub Releases.
  2. Installare ESPHome CLI e librerie Qt: 
    pip install esphome PyQt6 pyserial
  3. Estrarre l’archivio dell’app in una cartella dedicata.
  4. Avviare l’applicazione: 
    python main.py

2.2  Primo Avvio

Al primo avvio viene mostrata la finestra di scelta lingua:

È possibile cambiare lingua in qualsiasi momento: Impostazioni → Lingua.

2.3  Creare il primo progetto

  1. Menu File → Nuovo progetto.
  2. Selezionare la cartella di destinazione e digitare il nome del progetto.
  3. Nella scheda Impostazioni compilare:
    • Nome dispositivo
    • Scheda (es. esp32-devkit-v1)
    • SSID e Password Wi‑Fi
  4. Premere Aggiorna YAML per salvare le modifiche.
  5. Nella scheda Sensori trascinare un sensore sul canvas.
  6. Salvare (Ctrl+S) e compilare il firmware:
[Console]
INFO Compilazione in corso...
OK  Firmware compilato con successo
Attenzione: il nome progetto deve essere univoco nella rete locale per evitare conflitti mDNS (e.g. non utilizzare “test” se è già presente un dispositivo con identico hostname).
