1  Introduzione

1.1  Che cos’è ESPHomeGuiEasy

ESPHomeGuiEasy è un’applicazione desktop multipiattaforma che consente di creare, modificare e caricare firmware ESPHome senza la necessità di scrivere manualmente grandi porzioni di codice YAML. Pensata per maker, professionisti e appassionati di domotica, l’interfaccia grafica riduce significativamente la curva di apprendimento tipica di ESPHome, pur mantenendo accessibili le funzionalità avanzate per gli utenti esperti.

Nota: l’immagine img/main_window_overview.png deve mostrare la finestra principale completa, con evidenziati menubar, editor YAML e tab laterali.

1.2  Funzionalità principali

  • Editor YAML con evidenziazione sintattica, numeri di riga e ricerca rapida.
  • Canvas drag & drop per aggiungere sensori e creare automazioni visivamente.
  • Importazione ed esportazione del progetto in formato .zip con barra di avanzamento.
  • Compilazione firmware e caricamento su dispositivo USB o OTA con log in tempo reale.
  • Barre di progresso dedicate per operazioni lunghe (compilazione, flash, estrazione).
  • Menu moduli per abilitare rapidamente Logger, Web Server, MQTT, API, OTA, ecc.
  • Interfaccia multilingua (IT/EN/DE/ES) estendibile tramite semplici file JSON.
Suggerimento: se desideri contribuire a nuove traduzioni, consulta il Capitolo 7 “Supporto Multilingua”.
